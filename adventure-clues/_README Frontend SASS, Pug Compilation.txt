# Gulp and plugins instalation. Compilation of SASS and Pug, Livereload, Images Optimization

1. Install Node.js
3. Run "npm install".
4. Run 'gulp'. It will start server, SASS/Pug compilation, Livereload.
Production task - "gulp prod" - cleaning files, optimizes images, minify CSS etc.
